# Войсковой собор святого благоверного князя Александра Невского

Современный сайт для православного храма на базе WordPress с автоматическим импортом контента.

## 🚀 Быстрый старт

### 1. Активация темы
1. Зайдите в админку WordPress: `http://danil.local/wp-admin`
2. Перейдите в **Внешний вид → Темы**
3. Активируйте тему **"Church Kadence Child"**

### 2. Импорт контента (АВТОМАТИЧЕСКИ)

#### Вариант A: Python-скрипт (Рекомендуется)
```bash
cd /workspace
python3 scrape_and_import.py
```

Этот скрипт:
- Автоматически скачивает контент с alexander-nevskiysobor.ru
- Извлекает расписание, духовенство, контакты, историю
- Создает SQL файл `/workspace/app/sql/import_full_content.sql`
- Формирует красивые страницы с современным дизайном

#### Вариант B: Готовый SQL файл
Если скрипт уже отработал, просто импортируйте готовый файл:
```bash
# Через phpMyAdmin или команду mysql
mysql -u root wordpress < /workspace/app/sql/import_full_content.sql
```

### 3. Настройка меню
1. В админке: **Внешний вид → Меню**
2. Создайте новое меню "Главное меню"
3. Добавьте страницы:
   - Главная
   - Расписание богослужений
   - Духовенство собора
   - Контакты
   - История храма
4. Выберите расположение: **Главное меню (Primary)**
5. Сохраните

### 4. Настройка виджетов (опционально)
**Внешний вид → Виджеты:**
- Добавьте виджеты в "Боковая панель"
- Добавьте виджеты в "Подвал 1"

## 🆘 Решение проблем

### Контент не отображается
1. Проверьте, что тема активирована
2. Очистите кэш браузера
3. Проверьте, что SQL файл импортирован

### Стили не работают
1. Проверьте путь до style.css
2. Убедитесь, что theme URI правильный
3. Переактивируйте тему

### Ошибки при импорте
```bash
# Проверьте права доступа
chmod 755 /workspace/scrape_and_import.py

# Запустите с подробным выводом
python3 -u scrape_and_import.py
```
