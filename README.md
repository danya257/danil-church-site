# Войсковой собор святого благоверного князя Александра Невского

Этот сайт создан на базе WordPress для переноса информации с оригинального сайта церкви.

## Установка и настройка

1. Убедитесь, что Local by Flywheel запущен и сайт активен.
2. Зайдите в админку WordPress (http://danil.local/wp-admin) и активируйте тему "Church Kadence Child".
3. Откройте в браузере http://danil.local/import.php — это запустит импорт контента (страницы и посты).
4. В админке WordPress (Appearance > Menus) создайте меню "Главное меню" и добавьте ссылки на Главную, Расписание богослужений, Духовенство, Контакты.
5. Назначьте меню в "Главное меню" в настройках темы.

## Структура контента

- **Главная страница**: Новости и общая информация.
- **Расписание богослужений**: Времена служб.
- **Духовенство собора**: Информация о священниках.
- **Контакты**: Адрес, телефоны, социальные сети.
- **Новости**: Категория с постами о жизни прихода.

## Дизайн

Тема использует современный дизайн с градиентами, тенями и responsive layout. Цвета: темно-синий, золотой, красный.
