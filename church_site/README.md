### 🔧 Как изменить контент:

Все данные хранятся в базе данных `church_data.db`. Вы можете:

1. **Через Python:**
```python
import sqlite3
conn = sqlite3.connect('church_data.db')
c = conn.cursor()

# Изменить название сайта
c.execute("UPDATE options SET value = 'Новое название' WHERE key = 'site_name'")

# Добавить новость
c.execute("INSERT INTO news (title, date, excerpt, content) VALUES (?, ?, ?, ?)",
          ('Заголовок', '2024-04-20', 'Кратко', 'Полный текст'))

conn.commit()
conn.close()
```

2. **Через SQLite Browser:**
   - Скачайте [DB Browser for SQLite](https://sqlitebrowser.org/)
   - Откройте файл `church_data.db`
   - Редактируйте таблицы визуально
